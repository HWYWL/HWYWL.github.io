Semaphore（信号量）
Semaphore（信号量）是一个线程同步结构，用于在线程间传递信号，以避免出现信号丢失，或者像锁一样用于保护一个关键区域。

Semaphore（信号量）

内存模型，也说点简单的

栈（heap），函数加载的时候，为函数内部变量分配的空间。和父函数的内部变量和运行指针共享同一块区域。

函数运行时，new的空间，都是放在堆中的。
